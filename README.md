# Super Bowl Squares

A modern Super Bowl Squares app featuring yearly boards, name search, and winner highlighting, focused on clean UI, usability, and easy sharing.

## Features
- Year-based boards (current year shown by default)
- Search squares by name
- Highlight winning squares
- Clean, distraction-free UI
- Easy to share with participants

## Tech
- HTML, CSS, JavaScript
- Firebase Hosting

## Goals
This project focuses on UI/UX clarity while keeping the classic Super Bowl Squares experience simple and familiar.
